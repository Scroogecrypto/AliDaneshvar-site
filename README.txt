Ali Daneshvar Personal Website - Professional Version

How to test locally:
1. Unzip the folder.
2. Open index.html in any browser.

How to publish on Vercel:
1. Go to vercel.com and sign in.
2. Create a new project.
3. Upload this folder or connect it through GitHub.
4. After deployment, add the domain alidaneshvar.com in Vercel project settings.
5. Vercel will show the DNS records that must be added in Design.com.
